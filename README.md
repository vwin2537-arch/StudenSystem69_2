# ระบบรายงานการปฏิบัติงาน อุทยานแห่งชาติเอราวัณ

ระบบวางแผนวันปฏิบัติงาน เช็คชื่อ และรายงานผลการปฏิบัติงาน สำหรับเจ้าหน้าที่อุทยานแห่งชาติเอราวัณ

## Tech Stack
- **Backend:** Google Apps Script
- **Database:** Google Sheets
- **Storage:** Google Drive (รูปถ่าย)
- **Frontend:** HTML5, Tailwind CSS, SweetAlert2, Chart.js, DataTables
- **Font:** Kanit (Google Fonts)

### 3. สร้าง Google Apps Script Project
1. ไปที่ [script.google.com](https://script.google.com)
2. สร้างโปรเจกต์ใหม่
3. คัดลอกเนื้อหาจากไฟล์ `Code.gs` ไปวางในไฟล์ `Code.gs` ของโปรเจกต์

5. บันทึกโปรเจกต์

### 4. Deploy Web App
1. คลิก **Deploy > New deployment**
2. เลือก **Web app**
3. ตั้งค่า:
   - **Description:** ระบบรายงานการปฏิบัติงาน
   - **Execute as:** Me
   - **Who has access:** Anyone
4. คลิก **Deploy**
5. คัดลอก URL ที่ได้ไปเปิดในเบราว์เซอร์

### 5. ให้สิทธิ์ (Authorization)
- เมื่อ Deploy ครั้งแรก ระบบจะขอสิทธิ์เข้าถึง Google Sheets และ Google Drive
- คลิก **Review Permissions** > เลือกบัญชี Google > **Allow**

## แนวทางทดสอบระบบ

### ทดสอบฝั่ง Admin
1. **เข้าสู่ระบบ** ด้วย `admin` / `admin1234`
2. **สร้างผู้ใช้ใหม่** - เพิ่มข้อมูลเจ้าหน้าที่ในชีต Users โดยตรง (ID, Username, Password, Name, Role=User)
3. **สร้างรอบงาน** - ไปที่แท็บ "จัดการรอบงาน" > เลือกเจ้าหน้าที่ > กรอกวันเริ่มรอบ > สร้างรอบงาน
4. **อนุมัติแผน** - รอให้เจ้าหน้าที่ส่งแผน > ไปที่แท็บ "อนุมัติแผน" > กดอนุมัติ
5. **ตรวจสอบรายงาน** - ไปที่แท็บ "รายงาน" > ดูรายละเอียดและรูปถ่าย

### ทดสอบฝั่ง User (เจ้าหน้าที่)
1. **เข้าสู่ระบบ** ด้วยบัญชีเจ้าหน้าที่
2. **ส่งแผนวันทำงาน** - ไปที่แท็บ "แผนวันทำงาน" > เลือก 30 วัน > กดส่งแผน
3. **เช็คชื่อเข้างาน** - หลังแผนอนุมัติ > ไปที่แท็บ "ลงเวลาเข้างาน" > กดลงเวลา (ต้องเป็นวันที่อยู่ในแผนที่อนุมัติ)
4. **รายงานผล** - ไปที่แท็บ "รายงานผล" > กรอกงานที่ทำ > แนบรูปอย่างน้อย 3 รูป > กดส่ง
5. **ขอสลับวัน** - ไปที่แท็บ "ขอสลับวัน" > เลือกวันเดิม, วันใหม่, เหตุผล > ยื่นคำร้อง

## กฎธุรกิจสำคัญ
- 1 รอบงาน = **90 วัน**
- ต้องเลือกวันปฏิบัติงาน **30 วัน** ต่อรอบ
- เช็คชื่อได้เฉพาะ **วันที่อยู่ในแผนที่อนุมัติแล้ว** เท่านั้น
- ต้องแนบรูปถ่าย **อย่างน้อย 3 รูป** ต่อการรายงาน 1 ครั้ง
- รูปถูกบีบอัดฝั่ง Frontend ก่อนอัปโหลด (max 1200px, quality 70%)
